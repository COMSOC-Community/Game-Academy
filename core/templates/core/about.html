About the Game Academy

{% endblock %} {% block content %}

The Game Academy has been developed at the Institute for Logic, Language and Computation of the University of Amsterdam in the Computational Social Choice Group led by Ulle Endriss.

The core concept and design of the platform were conceived by Ulle Endriss and Simon Rey, with Simon Rey leading the actual implementation. Additional contributions were made by members of the group, including Adrian Haret and Julian Chingoma, who provided valuable input on various elements of the project.

The project is open-source. You can find the code in the GitHub repository COMSOC-Community/Game-Academy.
